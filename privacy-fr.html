Politique de Confidentialité

Inventory.AI

Introduction

Inventory.AI est une application pour gérer votre inventaire d'objets dans des boîtes. Cette politique décrit comment nous collectons, utilisons et protégeons vos données personnelles.

Données Collectées

Nous collectons : nom, prénom, adresse e-mail (pour le compte), photos d'objets et de boîtes, descriptions textuelles, préférences de l'app (thème, langue). Toutes les données sont exclusivement associées à votre compte.

Utilisation des Données

Vos données sont utilisées exclusivement pour fournir les services de l'app : gestion d'inventaire, analyse IA optionnelle, synchronisation cloud (si activée). Nous ne vendons ni ne partageons vos données personnelles avec des tiers à des fins commerciales.

Stockage des Images

Les photos d'objets sont stockées sur votre appareil en mode local. Si vous activez la synchronisation cloud, les images sont téléchargées sur Firebase Storage avec chiffrement. Seul vous pouvez accéder à vos images, qui restent complètement privées.

Synchronisation Cloud

La synchronisation cloud (achat optionnel) utilise Google Firebase pour stocker vos données. Les données sont protégées et accessibles uniquement depuis votre compte Firebase Authentication. Vous pouvez supprimer toutes les données cloud à tout moment en supprimant votre compte.

Services Tiers

Nous utilisons : Firebase (Google) pour l'authentification et le stockage ; Google AdMob pour la publicité (désactivable avec achat) ; Cloud Functions pour l'analyse IA d'images. Ces services peuvent collecter des données techniques selon leurs politiques de confidentialité.

Vos Droits

Vous avez le droit de : accéder à vos données, les modifier, les supprimer complètement (en supprimant le compte), exporter vos données. Pour exercer ces droits, utilisez les fonctions de l'app ou contactez-nous via le formulaire de contact.

Sécurité des Données

Nous utilisons Firebase Authentication pour protéger votre compte, le chiffrement pour les connexions, un stockage isolé pour chaque utilisateur. Les images sont compressées et stockées en toute sécurité. Cependant, aucun système n'est complètement sûr.

Contact

Pour des questions sur la confidentialité, contactez-nous via le formulaire de contact dans l'écran Autre de l'app.
