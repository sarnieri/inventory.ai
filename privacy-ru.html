Политика Конфиденциальности

Inventory.AI

Введение

Inventory.AI — это приложение для управления инвентарем предметов в коробках. Эта политика описывает, как мы собираем, используем и защищаем ваши личные данные.

Собираемые Данные

Мы собираем: имя, фамилию, адрес электронной почты (для аккаунта), фотографии предметов и коробок, текстовые описания, настройки приложения (тема, язык). Все данные связаны исключительно с вашим аккаунтом.

Использование Данных

Ваши данные используются исключительно для предоставления услуг приложения: управление инвентарем, опциональный AI-анализ, облачная синхронизация (если включена). Мы не продаем и не передаем ваши личные данные третьим лицам в коммерческих целях.

Хранение Изображений

Фотографии предметов хранятся на вашем устройстве в локальном режиме. Если вы включите облачную синхронизацию, изображения загружаются в Firebase Storage с шифрованием. Только вы можете получить доступ к вашим изображениям, которые остаются полностью конфиденциальными.

Облачная Синхронизация

Облачная синхронизация (опциональная покупка) использует Google Firebase для хранения ваших данных. Данные защищены и доступны только с вашей учетной записи Firebase Authentication. Вы можете удалить все облачные данные в любое время, удалив свой аккаунт.

Сторонние Сервисы

Мы используем: Firebase (Google) для аутентификации и хранения; Google AdMob для рекламы (отключается покупкой); Cloud Functions для AI-анализа изображений. Эти сервисы могут собирать технические данные согласно их политикам конфиденциальности.

Ваши Права

Вы имеете право: получать доступ к своим данным, изменять их, полностью удалять (удалив аккаунт), экспортировать свои данные. Чтобы воспользоваться этими правами, используйте функции приложения или свяжитесь с нами через контактную форму.

Безопасность Данных

Мы используем Firebase Authentication для защиты вашего аккаунта, шифрование для соединений, изолированное хранилище для каждого пользователя. Изображения сжимаются и хранятся безопасно. Однако ни одна система не является полностью безопасной.

Контакты

По вопросам конфиденциальности свяжитесь с нами через контактную форму в разделе Другое приложения.
